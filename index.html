🐳 About Me

I am a second-year Ph.D. student at The Chinese University of Hong Kong, supervised by Prof. Michael R.Lyu (IEEE/ACM/AAAS Fellow), where I am supported by Hong Kong PhD Fellowship (HKPFS). I obtained my M.S. (2021-2024) in Computer Science at Tsinghua University with Outstanding Graduate Award, supervised by Professor Yong Jiang and Professor Qing Li. I received my Bachelor's degree (2017-2021) from Wuhan University. My research interests mainly include: (1) Large Language Models: Code Intelligence, AI Safety. (2) Data Mining: User Behavior Modeling, Anomaly Detection, Recommendation and Graph Mining. (3) Computer Networks: Network Failure Localization, Network Function Virtualization and Programmable Switches.

🔥🔥The rapid advancements in generative AI, particular Multi-modal Large Language Models (MLLMs), now excelling in multimodal understanding and code generation. Recently, I mainly explore code intelligence of MLLMs in multimodal scenarios, e.g., UI code generation/repair, scientific plot code generation.

🔥🔥I created Awesome-Multimodal-LLM-for-Code repository and keep updating it. This is the first Awesome MLLM4Code repository and promotes code generation research under multi-modal scenarios. I also maintain the WebPAI (Web Development Powered by AI) project, aiming to build an AI platform for more reliable and practical automated webpage generation. If you find our work interesting, feel free to contact me for discussion or collaboration.
